# SmartQuizAI 🎓

SmartQuizAI is an AI-powered educational web application built with Flask that transforms PDF documents into interactive quizzes automatically. Students can upload study material in PDF format, generate AI-based questions, submit answers, and receive instant scoring with feedback.

## Features
- 📄 Upload PDF study material
- 🤖 AI-generated quiz questions
- ✅ Automatic answer evaluation
- 🔐 Student authentication system
- 📊 Quiz score tracking & history
- 🗄️ SQLite database integration
- ⚡ Built using Flask + Groq API

## Tech Stack
- Python
- Flask
- SQLite
- PDFPlumber
- Groq API (LLM)
- HTML/CSS (Frontend)
